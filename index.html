TimeRemind

Developed as a personal project to overcome 'time-blindness' commonly experienced by neurodiverse users, particularly those with ADHD.

This project introduced me to Wear OS development and unique UI elements eg overlays and round vs square watch faces. I also used Android's SharedPreferences API to store the user's chosen time interval so that it displayed correctly if the app was rebooted.

Coding TimeRemind deepened my understanding of activities and intents in Android development.

Written in Java using Android Studio and deployed to the Play Store as a free Wear OS standalone app.

November 2021

Badger

Badger was my university dissertation project and was developed to fill a gap in the existing Android productivity app market.

Badger stores reminders for the user which can be set, snoozed, and dismissed with voice control. The app uses Google Assistant and is designed to function handsfree for older, less tech-savvy users and users with executive dysfunctions as a result of ADHD.

Coded in Java, this project made me competent in Android development and extremeley comfortable with OOP. I also utilised several design patterns: builder, chain of repsonsibility, singleton.

A highlight is the custom conversational interface that I developed using TextToSpeech and SpeechRecognizer classes, which are executed sequentially by intents tagged with specific request codes, in order to build a user's entire reminder.
